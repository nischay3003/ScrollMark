# ScrollMark - Multi-LLM Bookmark Tool

Bookmark scroll positions across ChatGPT, Claude, Gemini, and any other websites!

## Features

This extension allows you to:
- **Bookmark scroll positions** on any website with a single click
- **Support for multiple LLMs**:
  - ChatGPT / OpenAI
  - Claude (claude.ai)
  - Google Gemini
  - Microsoft Copilot
  - Perplexity
  - HuggingFace Chat
  - DeepSeek
  - Groq
  - And any general website!

- **Filter bookmarks** by current chat or view all bookmarks
- **Platform badges** to see which platform each bookmark belongs to
- **Quick navigation** - click any bookmark to instantly scroll to that position
- **Easy management** - delete unwanted bookmarks with one click

## Installation

1. Clone or download this repository
2. Open your browser (Chrome/Brave)
3. Go to `chrome://extensions/` (or `brave://extensions/`)
4. Enable "Developer mode" in the top right
5. Click "Load unpacked" and select the extension folder

## How to Use

1. **Save a bookmark**:
   - Click the green **📌** button in the bottom-right corner of any supported website
   - Enter a name for your bookmark
   - Your bookmark is automatically saved with the platform info!

2. **View bookmarks**:
   - Click the extension icon in your toolbar
   - See bookmarks with platform badges
   - Toggle between "Current Chat" and "All Bookmarks"

3. **Navigate to a bookmark**:
   - Click on any bookmark name to instantly scroll to that position
   - In "All Bookmarks" mode, the page will open in a new tab and scroll automatically

4. **Delete a bookmark**:
   - Click the red **delete** icon next to any bookmark

## Supported Platforms

The extension works with:
- All LLM chat interfaces
- Any website with scrollable content
- Multiple simultaneous bookmarks per page
- Different conversations across different platforms

## What Makes It Special

- **Platform detection** - Automatically identifies which LLM you're using
- **URL preservation** - Stores full URL for bookmarks, allowing cross-platform navigation
- **Smart filtering** - "Current Chat" shows only bookmarks from your active conversation
- **Universal compatibility** - Works on any website, not just ChatGPT!

Enjoy seamless bookmarking across all your favorite AI platforms and websites!
